
# Día 1 - Linux básico

## Comandos aprendidos

- pwd = muestra la ruta actual
- whoami = muestra el usuario actual
- hostname = nombre del equipo
- ls = lista archivos visibles
- ls -la = lista todos los archivos (incluidos ocultos) con detalles
- cd = cambiar de directorio
- cd .. = subir un nivel
- cd ~ = ir al home
- mkdir = crear carpetas
- touch = crear archivos
- nano = editar archivos
- cat = ver contenido de archivos
- rm = borrar archivos
- rm -r = borrar carpetas

## Lo que entendí

Linux funciona con una estructura de directorios tipo árbol. 
Cada usuario tiene un espacio de trabajo y se pueden visualizar 
archivos, usuarios y configuraciones del sistema mediante comandos.

El archivo /etc/passwd contiene información de los usuarios del sistema.
